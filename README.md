Statistics and Dataset Fundamentals

About

This repository contains my study notes for **Statistics and Dataset Fundamentals** as part of my AI/ML Engineer training.

The notebook covers the basic concepts of statistics, different types of data, dataset structure, Python and Pandas data types, and real-world AI/ML examples. The notes are written in simple words with beginner-friendly examples.

Objectives

The main objectives of this sprint are:

- Build a strong foundation in statistics.
- Understand different types of data.
- Learn statistical terms used in AI/ML.
- Understand how datasets are structured.
- Prepare for future data preprocessing activities.
